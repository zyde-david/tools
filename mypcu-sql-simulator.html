                        วิธีการส่งออกเฉพาะหัวตาราง (Export Column Headers) จากโปรแกรม SQL
                        

                            ในการสืบค้นข้อมูลจริงจากระบบของคุณ (เช่น Navicat, DBeaver หรือ SQL Tool ใน HOSxP) เจ้าหน้าที่สามารถดึงเฉพาะหัวฟิลด์ของแฟ้ม m_EP หรือตารางอื่นๆ โดยรันคำสั่งด้านล่างนี้ และคัดลอกรายชื่อไปใช้ได้ทันที:

2

                    

                        การนำเข้าไฟล์ผัง XML พร้อมกันหลายตาราง (Multi-Table XML)
                        

                            คลิกที่ปุ่ม "นำเข้าไฟล์/โฟลเดอร์ XML (myPCU Schema)" สีเขียวสะดุดตาด้านบนสุด จากนั้นเลือกไฟล์ XML ของตารางต่างๆ (เช่น m_EP.xml, m_ADDRESS.xml) พร้อมกัน ระบบจะวิเคราะห์โครงสร้างและรวมฟิลด์เข้าพจนานุกรม Level 2 ให้คุณโดยอัตโนมัติ
                        

                    

                


                


                
                

                    
3

                    

                        การเพิ่ม แก้ไข ลบ คำอธิบายฟิลด์ข้อมูลสากล (Level 2 CRUD)
                        

                            เมื่อเลือกตารางจากแท็บ "คำอธิบายฟิลด์" คุณจะพบรายการฟิลด์และสัญรูปดินสอ/ถังขยะ คุณสามารถแก้ไขความหมาย หรือพิมพ์เพิ่มคอลัมน์มาตรฐานใหม่ด้วยตัวเองเพื่อป้องกันความเข้าใจผิดพลาดด้านข้อมูลคลินิก
